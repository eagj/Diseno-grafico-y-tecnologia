### PROGRAMAS GRATUITOS DE EDICIÓN DE VECTORES

A continuación listamos varias alternativas a Illustrator para la edición de vectores de forma gratuita:
![](https://l2eportal-lms.cclearning.accenture.com/contenidosSN/dise_grafico/es/dise_grafico_tecnologia/3_illustrator/content/img/large/img_31_1.svg)

[Inkscape](https://inkscape.org/es/ "Abre una nueva ventana​")
Es la alternativa más popular a Illustrator, ya que tiene muchas características similares a esta herramienta. Además, es compatible con los formatos de archivo más populares

![](https://l2eportal-lms.cclearning.accenture.com/contenidosSN/dise_grafico/es/dise_grafico_tecnologia/3_illustrator/content/img/large/img_31_2.png)

[Corel Vector​](https://www.coreldraw.com/en/product/vector/ "Abre una nueva ventana​")
Esta herramienta está disponible para usar _online_ e instalándola en el equipo. Cuenta con una interfaz sencilla para usuarios sin experiencia en la que tiene las herramientas básicas para la edición de gráficos.

![](https://l2eportal-lms.cclearning.accenture.com/contenidosSN/dise_grafico/es/dise_grafico_tecnologia/3_illustrator/content/img/large/img_31_3.png)

[Vectr](https://vectr.com/ "Abre una nueva ventana​")
Es un programa simple de usar que tiene las herramientas básicas para la edición de gráficos. Además, cuenta con la funcionalidad de compartir el progreso en tiempo real a través de una dirección de internet.

![](https://l2eportal-lms.cclearning.accenture.com/contenidosSN/dise_grafico/es/dise_grafico_tecnologia/3_illustrator/content/img/large/img_31_4.png)

[SVG-Edit](https://github.com/SVG-Edit/svgedit "Abre una nueva ventana​")
_Software_ de código abierto pionero entre las aplicaciones web de edición vectorial. Se puede usar en cualquier navegador moderno sin necesidad de registro. Se recomienda para usuarios con cierta experiencia en la edición vectorial

![](https://l2eportal-lms.cclearning.accenture.com/contenidosSN/dise_grafico/es/dise_grafico_tecnologia/3_illustrator/content/img/large/img_31_5.png)

[Figma](https://www.figma.com/ "Abre una nueva ventana​")
Esta herramienta está disponible para los sistemas operativos Windows, Linux, ChromeOS y macOS. Además, cuenta con herramientas propias para el diseño vectorial y tiene una gran funcionalidad colaborativa y diseño en grupo.
